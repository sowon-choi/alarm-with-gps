[문제 제기]
코로나19의 여파로 외출 빈도가 적어지며 외부 업무를 몰아서 처리함.
만약 외출시 할 일을 하지 못했다면 다시 나가거나 그 다음 외출로 미루는 등 굉장히 수고스러움.

[연구 개발 목표]
개발 프레임워크 사용
하이브리드 앱 구현
GPS api 활용 - 위치 서비스 기반 알림 설정

[연구 개발 내용]
1. 하이브리드 앱 구현
 1.1 OS호환
2. 개발 프레임워크 사용
 2.1 오픈 소스 크로스 플랫폼 앱 개발 프레임워크
3. GPS api 활용
 3.1 위치 서비스 기반 알림 및 메모 기능 구현

[연구 개발의 타당성]
기존의 시간 기반 알람 서비스는 고도화되고 있는 반면에 위치 기반 알람 서비스는 미비
알람에 메모 기능 추가
